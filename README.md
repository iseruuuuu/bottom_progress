## Getting started   

①
Add the dependency to your pubspec.yaml file.  
```
  dependencies:  
    flutter:  
      sdk: flutter  
    bottom_progress:   
```

②
Tap `Pub get` in your pubspec.yaml file.  

## Usage 
```
 BottomProgresss(
   // Number of page count.
   pointCount: 1,
   // Amount of bottom progress
   pageCount: 10,
   // page
   onColor: Colors.pink,
   // other page
   offColor: Colors.grey,
   // bottom progress size
   fontSize: 40,
 ),
```
